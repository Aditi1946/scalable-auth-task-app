# Scalable Auth Task App

A full-stack application with authentication, role-based access, and task management.

## 🚀 Features
- User Registration & Login (JWT)
- CRUD APIs for Tasks
- Secure Authentication
- React Frontend UI

## 🛠 Tech Stack
- Backend: Node.js, Express.js
- Database: MongoDB
- Frontend: React.js
- Auth: JWT, bcrypt

## 📦 Setup

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start

## 📌 API Endpoints
- POST /api/v1/auth/register
- POST /api/v1/auth/login
- GET /api/v1/tasks
- POST /api/v1/tasks
- PUT /api/v1/tasks/:id
- DELETE /api/v1/tasks/:id

## ⚡ Scalability
- Can be extended to microservices
- Redis caching support
- Load balancing ready
- Stateless JWT authentication
